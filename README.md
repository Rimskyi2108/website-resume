# Резюме-визитка

**Демо:** [Смотреть](https://rimskyi2108.github.io/website-resume/)

---

## 📌 Описание

Мой сайт-резюме, выполненный в тёмном стиле с эффектом стекла.  
Адаптивная вёрстка под все устройства, бургер-меню, плавный скролл.

---

## 🛠 Технологии

- HTML5
- CSS3 / SCSS
- JavaScript

---

## ✨ Особенности

- Тёмная тема с градиентным фоном
- Эффект стекла (glassmorphism) на карточках
- Адаптивная вёрстка (десктоп, планшет, мобилка)
- Бургер-меню с анимацией
- Плавный скролл по якорным ссылкам
- Кнопка Frost с эффектом заморозки
- Чистый, структурированный код

---

## 📱 Адаптивность

- Десктоп (до 2560px)
- Планшет (768–1024px)
- Мобильные устройства (до 768px)
